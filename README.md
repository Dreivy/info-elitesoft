# Info Elite Soft - Site Empresarial

Site institucional da Info Elite Soft - Soluções em Informática.

## 📁 Estrutura

```
info-elitesoft-site/
├── index.html       # Página principal
├── termos.html      # Termos de Uso
└── privacidade.html # Política de Privacidade
```

## 🚀 Como Hospedar no GitHub Pages (Gratuito)

### Passo 1: Criar repositório no GitHub
1. Acesse https://github.com/new
2. Nome do repositório: `info-elitesoft`
3. Marque como **Public**
4. Clique em **Create repository**

### Passo 2: Enviar os arquivos
1. No repositório criado, clique em **uploading an existing file**
2. Arraste os 3 arquivos HTML
3. Clique em **Commit changes**

### Passo 3: Ativar GitHub Pages
1. Vá em **Settings** → **Pages**
2. Em **Source**, selecione **main** branch
3. Clique em **Save**
4. Aguarde alguns minutos

### Passo 4: Acessar o site
Seu site estará em: `https://SEU-USUARIO.github.io/info-elitesoft`

## 📋 Para o Stripe

Use esta URL no campo "Site da empresa" no Stripe:
```
https://SEU-USUARIO.github.io/info-elitesoft
```
